# Заметки по репозиторию

GitHub-коннектор дал доступ на запись к `IvanRaketa/yandex-games-minimarket`, но часть файлов вроде `package.json`/`.gitignore` могла уже существовать в шаблоне репозитория. Поэтому временно добавлены `package.template.json` и `npm-scripts.json` как безопасная копия нужных команд.

Перед полноценной сборкой нужно привести это к одному `package.json`.
