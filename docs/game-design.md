# MiniMarket 24 — дизайн первой игры

## Коротко

`MiniMarket 24` — лёгкий казуальный бизнес-тайкун для Яндекс Игр. Игрок развивает маленький магазин: раскладывает товары, обслуживает покупателей, покупает улучшения и открывает новые отделы.

## Игровой цикл

1. Игрок раскладывает товар.
2. Покупатель приходит на кассу.
3. Игрок обслуживает покупателя.
4. Игрок получает деньги.
5. Деньги тратятся на улучшения.
6. Улучшения повышают доход, запас и скорость развития.
7. После выполнения дневной нормы начинается новый день.

## Почему этот жанр выбран первым

- Низкий риск по контентным требованиям Яндекс Игр.
- Можно сделать без тяжёлого 3D и уложиться в 100 МБ.
- Понятная монетизация через rewarded-рекламу.
- Хорошо подходит для мобильного управления.
- Можно постепенно добавлять контент без переписывания всей игры.

## Будущие rewarded-награды

- x2 прибыль за текущий день.
- Мгновенная поставка товара.
- Временный помощник кассира.
- Бесплатный сундук с деньгами.
- Ускорение покупателей на 3 минуты.

Rewarded-реклама не должна быть обязательной для прохождения.
