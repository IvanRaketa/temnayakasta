# Чек-лист требований Яндекс Игр

Этот файл нужен, чтобы перед публикацией пройти быструю самопроверку.

## Базовая структура

- [ ] В корне архива есть `index.html`.
- [ ] В названиях файлов и папок нет пробелов и русских символов.
- [ ] Распакованный размер игры меньше 100 МБ.
- [ ] Игра не требует внешнего сервера для основного геймплея.

## SDK

- [ ] Подключён Yandex Games SDK.
- [ ] `LoadingAPI.ready()` вызывается после готовности игры.
- [ ] Игра запускается даже без авторизации.
- [ ] Авторизация через Яндекс ID не обязательна.

## Геймплей

- [ ] Игра выглядит законченной, а не как тестовая сцена.
- [ ] Есть минимум 10 минут основного геймплея.
- [ ] Есть нарастающее развитие/сложность.
- [ ] Игрок понимает цель и управление без внешней инструкции.
- [ ] Прогресс сохраняется после перезагрузки страницы.

## Адаптация

- [ ] На телефоне интерфейс не обрезается.
- [ ] На ПК интерфейс не растягивается криво.
- [ ] Управление работает мышью и касанием.
- [ ] Долгое нажатие/правый клик не ломают геймплей.
- [ ] Игра корректно ведёт себя при сворачивании и возврате.

## Реклама и покупки

- [ ] Сторонней рекламы нет.
- [ ] Interstitial-реклама не появляется во время активного действия.
- [ ] Rewarded video запускается только добровольно.
- [ ] Награда за рекламу не является обязательной для прохождения.
- [ ] В первой версии покупки не используются.

## Контент

- [ ] Нет политики, войн, религии, реальных денег, крипты, наркотиков и другого рискованного контента.
- [ ] Все ассеты собственные или легально используемые.
- [ ] Иконка и обложка не являются обычным скриншотом.
- [ ] Скриншоты показывают реальный геймплей.
