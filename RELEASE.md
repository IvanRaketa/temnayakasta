# Как собрать архив для Яндекс Игр

## Проверка

```bash
npm install
npm run check
```

## Архив

```bash
npm run zip
```

Готовый файл появится здесь:

```text
release/minimarket-24-yandex-games.zip
```

В архиве `index.html` должен лежать в корне. Именно этот zip загружается в кабинет Яндекс Игр.

## Первая версия

Текущий MVP уже можно открыть как обычную HTML-страницу. Для локального запуска удобнее использовать:

```bash
npm install
npm run start
```

Дальше нужно добавить полноценную интеграцию SDK: облачные сохранения, rewarded-рекламу и паузу звука/игры на время рекламы.
