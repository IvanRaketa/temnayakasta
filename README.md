# MiniMarket 24

Первая HTML5-игра под Яндекс Игры.

## Статус

Проект готовится как лёгкая 2D/изометрическая казуальная игра: игрок развивает маленький магазин, обслуживает покупателей, покупает улучшения и возвращается за прогрессом.

## Цель первой версии

- пройти базовые требования Яндекс Игр;
- работать без обязательной регистрации;
- сохранять прогресс гостя локально;
- иметь минимум 10 минут геймплея;
- быть адаптированной под ПК и мобильные устройства;
- подготовиться к подключению Yandex Games SDK, рекламы и облачных сохранений.
